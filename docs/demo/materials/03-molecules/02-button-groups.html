--- name: button group title: button group notes: | Actions that logically belong together might be grouped by wrapping them in a container with .dc-btn-group. Each action button must have the modifier class .dc-btn--grouped Wrap several groups in a .dc-btn-group-row to keep them in one line. Each one must have the modifier class .dc-btn-group--in-row Please do not use the button group as tabs. ---
