--- name: button modifiers title: button modifiers notes: | Buttons can have modifier classes applied to them. Different modifier classes have different meanings, use them wisely. Primary: Triggers the primary action of the page it is placed on. Place only one primary button on each page/dialog. Destroy: Triggers a destructive action which can't be undone and might involve loss of data. Use these buttons sparingly and try to avoid undoable actions. Disabled: Any kind of button which is currently disabled looks like this. ---
