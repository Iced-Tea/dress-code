The Dress Code

The Dress Code is the style guide and UI toolkit for Zalando Brand Solutions products. It is a highly-modular design system for rapid web page development that contains different materials that can be assembled into more complex page layouts.

Usage

Start by adding the following files to your page:

<script src="https://cdnjs.cloudflare.com/ajax/libs/modernizr/2.8.3/modernizr.min.js"></script>
<link rel="stylesheet" href="bower_components/dress-code/dist/css/dress-code.css">
