--- notes: | Use these and nothing else, your users will appreciate it. ---

Get the font

Add this to your <head> tag.

<link href="http://fonts.googleapis.com/css?family=Ubuntu:300,500" rel="stylesheet" type="text/css">

Font Family

The one and only font declaration to be used.

font-family: Ubuntu, Helvetica, sans-serif;

Default Font Weight

Use this wherever you can, it's the default.

font-weight: 300;

Alternative Font Weight

Use to emphasize text in paragraphs using the <strong> tag and for labels.

font-weight: 500;

Font sizes

Headline H1 is 36px and the line-height is 48px

Headline H2 is 28px and the line-height is 36px

Headline H3 is 24px and the line-height is 36px

Headline H4 is 20px and the line-height is 24px

Size for normal text is 16px and the line-height is 24px

Size for small copy is 12px and the line-height is 16px
