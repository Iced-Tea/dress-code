Atoms

Buttons

Button

Triggers an action and must not be used for navigation.

Several instances of this component might be used on a page.

Anchor Tag 
<button class="dc-btn">Button Tag</button>
<a href class="dc-btn">Anchor Tag</a>

Button Primary

Triggers the primary action of the page it is placed on.

Place only one primary button on each page/dialog.

Primary Anchor Tag 
<button class="dc-btn dc-btn--primary">Primary Button Tag</button>
<a href class="dc-btn dc-btn--primary">Primary Anchor Tag</a>

Button Destroy

Triggers a destructive action (e. b. Delete, Take Offline), which often can't be undone and might involve loss of data.

Use these buttons sparingly and try to avoid undoable actions.

Negative Anchor Tag 
<button class="dc-btn dc-btn--destroy">Negative Button Tag</button>
<a href class="dc-btn dc-btn--destroy">Negative Anchor Tag</a>

Button Disabled

Any kind of button which is currently disabled looks like this.

disabled anchor tag 
<button class="dc-btn dc-btn--disabled">Disabled button tag</button>
<a href class="dc-btn dc-btn--disabled">disabled anchor tag</a>

Button Small

Each of the buttons has a smaller sibling which might be used where little space is available.

primary negative disabled

<a href class="dc-btn dc-btn--small">anchor tag</a>

Button Groups

Actions that logically belong together might be grouped with this component.

Button group rows can be used to combine several button groups in one line.

Alternatively, this component can be used as a radio button group where space is an issue.

Please do not use the button group as tabs.
