--- notes: | Block grids are great for making responsive pages. Use `.dc-block-grid--[small|medium|large|huge]-[1-12]` on the `.dc-row` to create a block grid. The columns inside only need `.dc-column` class applied to them. Block grids allow you to define how many columns per row you need based on the screen size. Block grids also facilitate you to add non-standard number of columns in a row like 5 columns. Example below shows that the columns below will be placed as "1 per row" on small screens, "3 per row" on meduim screens, and "5 per row" on large screens. ---
